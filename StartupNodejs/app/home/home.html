MEAN stands for:

MongoDB, Express.js, AngularJS, Node.js abbreviated as MEAN, these are application development pack by all Javascript. In this sample we are going to use MSSQL Server instead of MongoDb. With this application we can create a new user, show all the user, modify user data & also can able to delete the user through API build with Express from frontend using AngularJS.

Components:
  • MSSql - SQL database
  • Express - NodeJS Framework
  • Angular - JavaScript frameworks for Frontend
  • Node.js - JavaScript Execution environment (Server)
